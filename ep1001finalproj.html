My Final project

The idea behind my project is to create a automatic box that will have an automated opening and closing lid function , on the command of a push button.

Process of designing

  1. Planning and design thinking
  2. Designing the box and the varoous parts required using fusion 360
  3. 3D printing and 2D computed controlled machining of the parts
  4. Creating the electronics (PCB board)
  5. Interface and programming
  6. Assembling everything together

Components needed for the Project

  • 12 mm thickness wood - For pieces of the Box
  • 5mm wood for the box lid
  • Attiny 85 board
  • momentary push button - input
  • stepper motor - output
  • ISP programmer
  • Rack and pinion
  • motor stand

Computer Controlled Machining

With my design i have created in fusion 360 , i export it into the software Vcarve pro' to generate the toolpath needed to cut the pieces of my box

Programming and interface

Using my ISP board , i simply connected my attiny 85 board to the usb port of my laptop and use 'Arduino' to programme the board.

Using Vcarve pro. Once file has been generated , it is ready to cut.
